About the project Cryptonite - JQuery-AJAX RESTful API, Shahar Avshalom

  • The goal of the project is to access information from the world of virtual commerce (Bitcoin etc).
  • The home page contains information cards about virtual currencies (clicking "More info" will make a new call for information from the server, only if more than two minutes have passed since the last call).
  • Clicking the "Chart button" will add the currency to real-time reports, (up to 5 currencies can be selected).
  • The "Live Report screen" contains a graph of real-time selected currency rates.
