About Cryptonite project, Shahar Avshalom

The purpose of the project is to make information accessible from the world of virtual trade.
The Home screen contains information cards about the coins.
Clicking the chart button will add the coin to the Live Reports charts, (You can select up to 5 coins).
The Live Report screen contains a graph of the selected coins.
